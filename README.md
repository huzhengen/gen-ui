# GEN-VUE-UI

GEN-VUE-UI is a simple Vue UI library.

### Install

```
npm install gen-vue-ui
```

or

```
yarn add gen-vue-ui
```

### Use Vue SFC

```
<template>
  <div>
    <Button>click</Button>
  </div>
</template>
<script>
import {Button, Tabs, Tab, Switch, Dialog, openDialog} from "gen-vue-ui"
export default {
  components: {Button}
}
</script>
```
