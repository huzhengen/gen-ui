# GEN-UI

GEN UI is a simple Vue UI library.

### Install

```
npm install gen-ui
```

or

```
yarn add gen-ui
```

### Use Vue SFC

```
<template>
  <div>
    <Button>click</Button>
  </div>
</template>
<script>
import {Button, Tabs, Tab, Switch, Dialog, openDialog} from "gen-ui"
export default {
  components: {Button}
}
</script>
```
